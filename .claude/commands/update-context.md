Please update `.claude/context.md` with today's session changes.

Follow this process:

1. **Read the current context.md file** to see the existing format

2. **Add a new session entry** to the "Recent Changes" section with today's date

3. **Include the following in the new session entry:**
   - **Changes Made:** List what was changed, created, or deleted (with ✅ checkmarks)
   - **Files Modified:** List all files that were created, edited, or deleted
   - **Decisions Made:** Any important decisions or design choices
   - **Discovered Issues:** Any new gotchas, bugs, or edge cases found

4. **Update "Current State" section** if needed:
   - Mark tasks as complete if done
   - Add new tasks if discovered
   - Update "What's Working" or "What Needs Work" sections

5. **Update "Ongoing Tasks" section** if needed:
   - Mark completed tasks
   - Add new tasks discovered during session
   - Update priorities if changed

6. **Add any new "Common Gotchas"** discovered during this session

7. **Update the "Last Updated" date** at the top of the file to today's date

8. **Show me a summary** of what you added so I can review before you save

After showing the summary, ask if I want to proceed with the update.

---

## Example Session Entry Format:

```markdown
### Session: 2025-11-02 (Brief Description)

**Changes Made:**
1. ✅ Did something important
   - Detail 1
   - Detail 2
2. ✅ Created new feature
   - Detail

**Files Modified:**
- Created: `new-file.py` (description)
- Modified: `existing-file.py` (what changed)
- Deleted: `old-file.py` (why)

**Decisions Made:**
- Decided to use approach X instead of Y because...
- Set configuration Z to value W

**Discovered Issues:**
- Found edge case where...
- Noticed performance issue with...
```
